# Quick Start Guide

Get up and running with OG Preview in 5 minutes!

## Installation

1. **Download or clone the repository**
   ```bash
   git clone https://github.com/BlindTrevor/OG-Preview.git
   ```

2. **Upload to WordPress**
   - Copy the `OG-Preview` folder to `/wp-content/plugins/`
   - Or rename to `og-preview` for a cleaner name

3. **Activate**
   - Go to WordPress Admin > Plugins
   - Find "OG Preview"
   - Click "Activate"

Done! The plugin is now active.

## First Use

### In WordPress Editor

1. Create or edit any post/page
2. Look for "Social Media Preview" box in the right sidebar
3. Click platform tabs (Facebook, Twitter, WhatsApp, LinkedIn)
4. See your preview!
5. Click "Refresh Preview" after making changes

### In Elementor

1. Edit any page with Elementor
2. Look for the pink floating button (bottom-right)
3. Click it to open the preview panel
4. Switch between platforms
5. Click "Refresh Preview" to update

## Configuration

Go to **Settings > OG Preview** to enable/disable specific platforms.

## What You'll See

The plugin shows exactly how your post will look when shared on:
- **Facebook** - Large image, title, description, domain
- **Twitter** - Card style with image and text
- **WhatsApp** - Compact horizontal layout
- **LinkedIn** - Professional card style

## Common Questions

**Q: Where's the meta box?**
A: Look in the right sidebar when editing a post/page.

**Q: I don't see the Elementor button?**
A: Make sure Elementor is installed and you're in the Elementor editor.

**Q: Can I customize the OG tags?**
A: Yes! Use Yoast SEO, Rank Math, or custom meta fields.

**Q: Does it work with custom post types?**
A: Yes! It works with all public post types.

**Q: Is it safe to use?**
A: Yes! All outputs are escaped, inputs sanitized, and AJAX is nonce-protected.
