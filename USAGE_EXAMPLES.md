# OG Preview Plugin - Usage Examples

## Example 1: Using the Meta Box in WordPress Editor

When editing a post or page in the classic WordPress editor, you'll see a "Social Media Preview" meta box in the sidebar. This box shows:

1. **Platform Tabs**: Switch between Facebook, Twitter, WhatsApp, and LinkedIn
2. **Live Preview**: See exactly how your content will appear when shared
3. **Refresh Button**: Update the preview after making changes to title, content, or featured image

## Example 2: Using with Elementor

When editing a page in Elementor:

1. Look for the pink/magenta floating button in the bottom-right corner (share icon)
2. Click it to open a slide-out panel
3. Navigate between platform previews
4. The preview updates in real-time based on your page content

## Example 3: What Gets Previewed

The plugin automatically extracts:

- **Title**: From post title, or SEO plugin OG title if set
- **Description**: From post excerpt, content, or SEO plugin OG description
- **Image**: From featured image, SEO plugin OG image, or site logo
- **URL**: The post permalink
- **Site Name**: Your WordPress site name

## Example 4: Integration with SEO Plugins

If you use Yoast SEO or Rank Math:

1. The plugin will automatically use your configured OG tags
2. No additional configuration needed
3. Previews will match what SEO plugins would generate

## Example 5: Admin Settings

Go to **Settings > OG Preview** to:

- Enable/disable specific platforms (Facebook, Twitter, WhatsApp, LinkedIn)
- Only show platforms you care about

## Preview Platform Details

### Facebook
- Large image (1200x630 recommended)
- Domain shown at top
- Title in bold
- Description below title
- Classic Facebook link card style

### Twitter
- Large image preview
- Title in bold
- Description
- Domain at bottom
- Twitter card style

### WhatsApp
- Compact horizontal layout
- Site name and title on left
- Small thumbnail (80x80) on right
- Minimal style matching WhatsApp link previews

### LinkedIn
- Professional appearance
- Large image
- Bold title
- Domain below
- Business-focused presentation

## Best Practices

1. **Image Size**: Use 1200x630px for optimal display across platforms
2. **Title Length**: Keep under 60 characters for best display
3. **Description**: Aim for 150-160 characters
4. **Test Before Publishing**: Always check previews before publishing
5. **Update Regularly**: Use refresh button after making changes
